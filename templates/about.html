About The Array

Calling all CCI students!

Do you struggle to choose courses that teach what you want to know? Are you looking to be better prepared for what the industry demands?

Introducing:
The Array.
For UNCC Students in the process of choosing a course of study, The Array is a website that provides information on which software and programming languages that the various classes in UNCC use, how often they are used, and various software uses. Unlike the UNCC Software Webstore, The Array explains the associations between UNCC classes, programming languages, and software, giving users the agency to seek classes that use software and languages they wish to become more familiar with.
Courses

Conveniently browse CCI courses, view descriptions, and explore their requirements.

Software

See what software works with the programming languages you have in mind.

Languages

Explore languages by course requirements, and get help setting them up with online guides.

Search

A comprehensive search function allows you to effortlessly query all content available on The Array.

Professors and TAs

CCI faculty and assistants can update course details, write guides, and more with simple edit interfaces.
